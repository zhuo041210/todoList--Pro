## ✨ 功能特性

### 📝 任务管理
- **添加任务**：输入框键入内容，按 Enter 或点击「添加」按钮即可创建
- **删除任务**：鼠标悬停显示删除按钮，点击并确认后移除（防误删）
- **批量删除**：一键清空所有「已完成」的任务，并显示删除数量确认
- **编辑任务**：双击任务文本进入编辑模式，支持回车确认 / ESC 取消
- **标记完成**：点击任务前方的圆框切换完成状态（✅ 已完成 / ⬜ 未完成）
- **全选切换**：一键将全部任务标记为「已完成」或「未完成」

### 🔍 搜索与筛选
- **实时搜索**：输入关键词，任务列表即时过滤匹配项
- **搜索高亮**：搜索结果动态更新，不影响原始数据

### 🎨 数据可视化
- **进度条**：直观展示总体完成进度，颜色随完成率变化（红 → 橙 → 蓝 → 绿）
- **饼图统计**：鼠标悬停进度条弹出详情窗口，展示已完成 / 未完成的比例
- **数字统计**：顶部实时显示「未完成任务数量」

### 🖱️ 交互体验
- **拖拽排序**：通过拖拽自由调整任务顺序（搜索时自动禁用，避免数据错乱）
- **双击编辑**：快速修改任务内容，操作流畅
- **键盘快捷键**：
  - `Enter`：添加任务 / 确认编辑
  - `ESC`：取消编辑
- **防误操作**：删除任务时弹出二次确认对话框

### ⏰ 时间与状态
- **实时时钟**：显示当前时间（时:分:秒），每秒自动更新
- **日期显示**：展示「年月日 + 星期几」（如：2026年06月25日 星期四）
- **工作日识别**：自动判断当天是否为工作日，标题颜色动态变化（工作日红色 / 休息日默认色）

### 💾 数据持久化
- **本地存储**：所有任务数据自动保存到浏览器 localStorage
- **刷新不丢失**：页面刷新或关闭后，数据依然保留

### 📦 附加功能
- **空状态提示**：无任务时展示温馨占位文案「✨ 没有待办 · 享受宁静吧」
- **任务计数**：实时显示当前未完成的任务数量
- **响应式布局**：适配不同屏幕尺寸（PC端）
- **平滑动画**：组件进入/离开均带过渡动画，提升视觉体验


## 🛠️ 技术实现

| 模块 | 技术方案 |
|------|----------|
| 前端框架 | Vue 2.x |
| 状态管理 | Vuex（全局状态 + localStorage 同步） |
| UI 组件库 | Element UI（消息提示、确认弹窗） |
| 数据可视化 | ECharts（饼图） |
| 拖拽功能 | vuedraggable |
| 时间处理 | moment.js |
| 数据持久化 | localStorage |
| 构建工具 | Vue CLI |
| 版本控制 | Git + GitHub |
| 持续部署 | Netlify（自动构建发布） |
